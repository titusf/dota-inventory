Join The Team

We are currently a very small team of developers and designers from around the world who are always looking for more support. If you have a spare couple of hours a week and think the project looks interesting, the following roles are currently available:

AngularJS Developer: This is where the presentation happens. Steam provides a lot of interesting data and we use AngularJS to transform it and bring it all together for the user to see. Fancy a fun little side-project?

PHP Developer: Our back-end is built with PHP and provides the core data through an API that we maintain. The back-end is still very much a work-in-progress with a lot of potential improvements to be made.

Designer: The site will always need a designer's touch. We want the site to look and feel great so get on board and add some flair!
